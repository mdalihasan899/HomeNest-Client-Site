HomeNest - Real Estate Listing Portal

HomeNest is a modern real estate listing platform where users can explore, buy, rent, and manage property listings with ease. It allows property owners to post their available properties for sale or rent and enables users to browse, search, sort, and review properties dynamically.  

Live Site URL:  https://home-nest-ali-hasan.netlify.app

Features:
1. Fetch and display properties from MongoDB with real-time updates and sorting by price/date.  
2. Email/Password and Google login with Firebase Authentication.  
3. Add, update, and delete your own property listings with live toast notifications.  
4. Authenticated users can rate and review properties, with all reviews visible on property details.  
5. Certain routes like *Add Property*, *My Properties*, and *My Ratings* are accessible only after login.  
6. Toggle theme support for better user experience.  
7. Easily search by property name and sort by price or date.  
8. Works seamlessly on mobile, tablet, and desktop devices.  
9. Built with Tailwind CSS, React, and animation effects using AOS for smooth transitions.
